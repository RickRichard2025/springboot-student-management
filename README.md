# Student Management API (Spring Boot)

## Overview
This project is a REST API built with Spring Boot that allows managing a list of students using CRUD operations.

## Technologies
- Java
- Spring Boot
- Spring Web

## Features
- Get all students
- Get student by email
- Create student
- Update student (full and partial)
- Delete student

## How to Run
1. Clone the repository
2. Run the application
3. Test endpoints with Postman
